Keyboard Shortcuts

Pressing numbers 1-9 and operators (parentheses, arithmetic symbols [+,-,*,/,], and carets) will add that symbol to the input area.

Pressing "=" or ENTER will execute the equation. If you are in algebra mode, "=" behaves like an operator.

Pressing "h" opens the help page. "s" opens settings.

SHIFT + 1 opens the standard keypad. SHIFT + 2 opens the advanced keypad

Pressing "p" adds the pi symbol

Pressing "f" puts Coyote in factoring mode. "a" activates algebra, and "r" activates square root simplification

"c" clears the current function and all saved modes.

"r" completely resets the calculator session.
